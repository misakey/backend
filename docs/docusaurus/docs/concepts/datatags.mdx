> A datatag is free text attached to an organization and some data. It allows the data type identification, for easier retrieval of data but also to enable data sharing consent.

#### Datatags and Organizations

- A datatag is linked to an [organization](organizations.mdx) and created within its context by administrators.
- A datatag can be attached to a [box](boxes.mdx) to identify the kind of data the box contains. Example of datatags: _invoices_, _purchases_, _contracts_, _hobbies_, _any_tag_making_sense_for_your_needs_...

Check out [this guide](/guides/store-data-for-your-org.mdx) to create datatag and assign them to boxes for your organization!
